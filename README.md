=============================================================================

Tencent Weibo Android SDK, not use commons-httpclient-3.0.1.jar

腾讯微博官方使用了比较旧的httpclient-3.0.1,使用commons-httpclient-3.0.1.jar使apk包大,而且httpclient-3.0.1版本太低.
此版本适用Android自带动httpclient.
